# gizmo

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run watch
```

### Register
```
POST /api/v1/user/register
{
    email: "",
    password: "",
    phone: "",
    name: ""
  }

```

### Login
```
POST /api/v1/auth/login
{
    email: "",
    password: "",
  }

```

### Add property
```
POST /api/v1/property/add-property
{
    email: "",
    password: "",
  }

```
